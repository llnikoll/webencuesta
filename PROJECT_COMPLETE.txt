🚀 EMPEZAR EN 5 MINUTOS

  1. Instalar dependencias:
     $ npm install

  2. Configurar base de datos en .env.local:
     DATABASE_URL="postgresql://usuario:contraseña@localhost:5432/webencuesta"

  3. Crear base de datos:
     $ npm run prisma:migrate

  4. Crear datos iniciales:
     $ npm run prisma:seed

  5. Iniciar servidor:
     $ npm run dev

  🌐 Accede a: http://localhost:3000

🔐 SEGURIDAD MULTINIVEL

  Capa 1: IP Hash (SHA256)
  ├─ Hash irreversible de IP del cliente
  ├─ Previene votos desde misma conexión
  └─ ✓ Implementado

  Capa 2: Device Fingerprint
  ├─ User Agent + Navegador + Timezone + Canvas
  ├─ Imposible de falsificar
  └─ ✓ Implementado

  Capa 3: Cookies HttpOnly
  ├─ No accesible desde JavaScript
  ├─ Automática en cada request
  └─ ✓ Implementado

  Capa 4: User Agent Tracking
  ├─ Registro de navegador/SO
  ├─ Para auditoría y detección
  └─ ✓ Implementado

  + Rate Limiting (3 intentos/60s)
  + Sanitización de inputs (XSS)
  + Protección CSRF
  + Validación backend completa

🔄 FLUJO DE VOTACIÓN

  Usuario
    ↓
  [1] Accede a http://localhost:3000
    ↓
  [2] Selecciona encuesta
    ↓
  [3] Elige una opción
    ↓
  [4] Presiona "Votar"
    ↓

    ↓

    ├─ Verifica no hay duplicados
    └─ Registra en BD
    ↓
  [7] "✓ Tu voto fue registrado correctamente"
    ↓
  [8] Intenta votar de nuevo
    ├─ Desde mismo dispositivo: ❌ "Ya votaste"
    └─ Desde otro dispositivo: ✅ Puede votar
    ↓
  [9] Espera 72 horas (Countdown)
    ↓
  [10] Se revelan automáticamente los resultados
